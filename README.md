# Computational Physics Lab Portfolio

This repository contains computational analysis and modeling projects developed from Physics laboratory courses at New York University Shanghai, along with independent extensions of experimental work.

The projects focus on bridging **experimental physics** with **computational data analysis**, using Python to extract physical insights from real laboratory measurements.

---

## 🧪 Origin of the Projects

All projects in this repository are based on data collected from undergraduate Physics laboratory courses at NYU Shanghai. These labs include experiments in classical mechanics, optics, and electromagnetism.

Instead of limiting the work to traditional lab reports, each experiment has been extended into a computational framework using Python for:

- Data analysis and visualization  
- Model verification and curve fitting  
- Numerical simulation of physical systems  
- Uncertainty and error analysis  

---

## 📊 Projects Included

### 1. Snell’s Law – Optical Refraction Analysis
Experimental verification of Snell’s Law using angle measurements from optics labs.  
Includes:
- Refractive index calculation from experimental data  
- Linear regression of sin(θ₁) vs sin(θ₂)  
- Visualization of optical refraction behavior  



---

## 🧠 Methods & Tools

- Python (NumPy, Matplotlib, SciPy)  
- Data fitting and regression analysis  
- Monte Carlo simulations  
- Numerical modeling of physical systems  
- Error propagation and uncertainty analysis  

---

## 🎯 Objective

The goal of this repository is to extend traditional undergraduate Physics laboratory work into a computational framework, demonstrating the ability to:

- Translate experimental data into physical models  
- Use computational tools to analyze and visualize results  
- Develop deeper physical intuition through simulation
